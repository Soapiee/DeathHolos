## Introduction
Spawn a hologram of your design, whenever a player dies!
Add placeholders to display their cause of death, and more...​

## Compatibility
- Tested engines: Spigot or Paper.
- Version: 1.16 and 1.21.11

## Dependencies
There is one **OPTIONAL** dependency
1. [Placeholder API](https://www.spigotmc.org/resources/placeholderapi.6245/)

## Installation
1. [Download](https://www.spigotmc.org/resources/death-holograms-1-16-26-1.133766/) the plugin.
2. Place the .jar file inside your "/Plugins" folder on your Minecraft server.
3. Start the server.
4. Edit the "config.yml" to set how the holograms look, and add new ones!
5. Run the reload command ("/adeathholos reload") and die in game.
